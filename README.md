# Mortal-Kombat-Multiverse
Mortal Kombat Multiverse Mugen Fighting Game Designed by myself, Jesse Middleton [SubScorp]

I made this game to be the fighting game of all fighting games, where all the best ones are in one game! Thus this project became: MORTAL KOMBAT MULTIVERSE. This game consists of characters from Street Fighter, Tekken, Power Rangers, Double Dragon, Primal Rage, Marvel, DC, King of Fighters, Fatal Fury, World Heroes, Spawn, Dark Stalkers, Killer Instinct, Teenage Mutant Ninja Turtles, X-Men, and Mortal Kombat.

This is built off of the  Mugen Game Engine. Runs on PC and Android via Winlator Emulator.
